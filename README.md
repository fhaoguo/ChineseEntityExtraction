# 神经网络中文实体抽取项目
+ 简介：基于Tensorflow 1.15深度学习框架，使用谷歌的BERT模型在BiLSTM-CRF模型上进一步训练，进行中文实体识别。
+ 训练数据来源：https://github.com/zjy-ucas/ChineseNER
